# 要删除的短语

## 开场白废话

这些预告式短语直接删掉，直接陈述内容。

- "事情是这样的："
- "关键在于——"
- "有意思的是——"
- "问题在于——"
- "原因是这样的——"
- "令人不安的真相是"
- "事实证明"
- "真正的XX是"
- "让我把话说清楚"
- "真相是，"
- "我再说一遍："
- "说实话"
- "我们来谈谈——"
- "我觉得有趣的是——"
- "不过问题在于——"

任何"事情是/关键是/原因是"结构的句子都是在绕弯子。删掉它，直接说重点。

## 强调惯用语

这些词不增加任何含义，删掉。

- "句号。" / "完了。"
- "好好想想吧。"
- "这之所以重要，是因为"
- "不要搞错"
- "这就是为什么这很重要"

## 商务黑话

用大白话替换。

| 避免使用 | 改用 |
|----------|------|
| 应对（挑战） | 处理、解决 |
| 拆解（分析） | 解释、审视 |
| 拥抱（不确定性） | 接受、正视 |
| 格局（背景） | 情况、领域 |
| 颠覆性的 | 重要的、显著的 |
| 加码 | 加大投入、坚持 |
| 深度剖析 | 分析、研究 |
| 退一步看 | 重新考虑 |
| 展望未来 | 接下来、今后 |
| 回头再说 | 回到、重提 |
| 达成共识 | 意见一致、同意 |

## 副词

所有副词全部删掉。不用"地"字修饰词。不要软化词，不要强化词，不要模糊词。

特别注意这些：

- "真的"
- "只是"
- "简直"
- "真心"
- "老实说"
- "简单的"
- "其实"
- "深深地"
- "真正地"
- "根本上"
- "本质上"
- "不可避免地"
- "有趣的是"
- "重要的是"
- "关键的是"

同时删掉这些填充短语：

- "从本质上讲"
- "在当今的XX时代"
- "值得注意的是"
- "说到底"
- "当谈到——"
- "在一个XX的世界里"
- "现实是"

## 元评论

删除自言自语式的旁白。文章应该自己流动，不要预告自己的结构。

- "提示："
- "剧透："
- "你其实已经知道这一点，但是"
- "不过那是另一篇文章了"
- "XX是特性，不是缺陷"
- "伪装成"
- "本文剩余部分将解释……"
- "让我带你过一遍……"
- "在本节中，我们将……"
- "我们将会看到……"
- "我想探讨……"

## 表演式强调

虚假的亲近感或刻意制造的真诚：

- "悄悄渗入"
- "我保证"
- "它们确实存在，我保证"

## 告知而非展示

宣告难度或重要性，而非直接呈现：

- "这真的很难"
- "这才是领导力的真正模样"
- "这才是XX的真正模样"
- "才是真正重要的"

## 模糊断言

那些宣称重要性却不指出具体内容的句子。全部删掉。

- "原因的是结构性的"
- "影响是深远的"
- "这是最深层的问题"
- "风险很高"
- "后果是真实的"

如果一句话只说某件事重要/深刻/结构性，却不展示具体是什么，要么删掉，要么用具体内容替换。
