# 要避免的结构

## 二元对比

这种结构制造虚假的戏剧感。直接陈述观点。

| 模式 | 问题 |
|------|------|
| "不是因为X。而是因为Y。" | 提前预设的反转 |
| "X不是问题。Y才是。" | 公式化的观点翻转 |
| "答案不是X。而是Y。" | 可预测的转折 |
| "感觉像是X。实际上却是Y。" | 设问/揭晓的陈词滥调 |
| "问题不在于X。而在于Y。" | 修辞上的刻意误导 |
| "不是X。而是Y。" | 机械对比 |
| "不是这个。是那个。" | 同样的公式，换了词 |
| "不再是X，而变成了Y" | 虚假的转变弧线 |
| "不意味着X，实际上是Y" | 否定再断言的习惯 |
| "是关于X而非Y" | 虚假的区分 |
| "不仅是X，也是Y" | 附加性回避 |

**正确做法：** 直接说Y。"问题是Y。""Y在这里很重要。"完全放弃否定部分。

## 否定列举

先说不是什么，再说是什么。一种修辞上的脱衣舞。

| 模式 | 问题 |
|------|------|
| "不是一个X……不是一个Y……一个Z。" | 通过否定制造戏剧性铺垫 |
| "那不是X。那不是Y。那是Z。" | 同样的结构，过去时 |

**正确做法：** 直接说Z。读者不需要跑道。

## 戏剧化碎片

用短句碎片制造强调，读起来像刻意制造深度。

| 模式 | 问题 |
|------|------|
| "[名词]。就这样。这就是[某物]。" | 表演式的简单 |
| "X。还有Y。还有Z。" | 断奏式戏剧 |
| "这解锁了一些东西。[单词]。" | 人造的顿悟 |

**正确做法：** 完整的句子。相信内容胜于包装。

## 修辞设问

这些是在预告洞见，而非传递洞见。

| 模式 | 问题 |
|------|------|
| "如果[重新定义]会怎样？" | 苏格拉底式的装腔作势 |
| "我的意思是这样的：" | 多余的预告 |
| "你想一想：" | 居高临下的提示 |
| "这也没关系。" | 不必要的许可 |

**正确做法：** 直接说出观点。让读者自己得出结论。

## 公式化结构

| 模式 | 问题 |
|------|------|
| "到X的时候，我已经Y了。" | 叙事模板 |
| "一个并非Y的X" | 间接表达。直接说"X坏了" |

## 虚假主体性

给无生命的东西配上人类动词。投诉不会"变成"解决方案。赌注不会"生死未卜"。决策不会"浮现"。是人做了这些事。AI喜欢这样写，因为可以避免指出动作者是谁。

| 模式 | 问题 |
|------|------|
| "一个投诉变成了解决方案" | 投诉什么都没做。是有人解决了它。 |
| "赌注在几天内决定生死" | 赌注没有寿命。是有人杀掉了项目或发布了产品。 |
| "决策浮现了" | 决策不会自己浮现。是有人做了决定。 |
| "文化转变了" | 文化不会自己转变。是人们改变了行为。 |
| "对话转向了" | 对话不会自己转向。是有人在引导。 |
| "数据告诉我们" | 数据就摆在那里。是有人读了并得出结论。 |
| "市场奖励了" | 市场不会奖励。是买家付了钱。 |

**正确做法：** 说出那个人。"团队当周就修好了"胜过"投诉变成了解决方案"。如果无法指定具体的人，用"你"让读者代入。

## 旁观者视角

漂浮在场景上方，而非让读者进入场景。

| 模式 | 问题 |
|------|------|
| "没有人设计过这个。" | 游离的观察 |
| "这之所以发生是因为……" | 授课者语气 |
| "这就是为什么……" | 同上 |
| "人们倾向于……" | 沙发上的社会学家的口吻 |

**正确做法：** 让读者进入场景。"你不会某天坐下来然后决定……"胜过"没有人设计过这个。"

## 被动语态

每句话都需要一个主语在做某事。被动语态隐藏动作者，耗尽能量。

| 模式 | 修正 |
|------|------|
| "X被创建了" | 说出是谁创建的 |
| "人们认为" | 说出是谁认为的 |
| "错误被犯下了" | 说出是谁犯的 |
| "决定被做出了" | 说出是谁决定的 |

**正确做法：** 找到动作者。把他们放在句子的最前面。

## 要避免的句子开头

| 模式 | 修正 |
|------|------|
| 以"什么、何时、哪里、哪个、谁、为什么、如何"开头的句子 | 重构。以主语或动词开头。 |
| 以"所以"开头的段落 | 以内容开头 |
| 以"听着，"开头的句子 | 删除 |

以疑问词开头成了拐杖。"让这件事变难的是……"改成"限制条件是……"或更好的是，说出那个具体的限制条件。

## 节奏模式

| 模式 | 修正 |
|------|------|
| 三项式排比 | 用两项或一项 |
| 问题立刻被回答 | 让问题有呼吸空间，或删掉问题 |
| 每段都以短促金句结尾 | 变化结尾方式 |
| 破折号 | 删除。用逗号或句号。完全不要用破折号。 |
| 断奏式碎片 | 不要堆叠短促有力的句子 |
| "并非总是。并非完美。" | 伪装成安慰的回避 |

## 词语模式

| 模式 | 问题 |
|------|------|
| 懒惰的极端词（每个、总是、从不、每个人、所有人、没有人） | 虚假的权威感。用具体内容代替笼统断言。 |
| 所有副词（"地"字修饰词、"真的""只是""简直""真心""老实说""简单的""其实"） | 空洞的强调。详见 phrases_CN.md 的完整列表。 |
