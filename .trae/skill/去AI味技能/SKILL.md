# Stop Slop（去AI味）

消除文章中可预测的AI写作模式。

## 核心规则

1. **删除填充短语。** 去除开场白废话、强调惯用语和所有副词。详见 [参考/短语.md](参考/短语.md)。

2. **打破公式化结构。** 避免二元对比、否定列举、戏剧化碎片、修辞设问、虚假主体性。详见 [参考/结构.md](参考/结构.md)。

3. **使用主动语态。** 每句话必须有一个人类主语在做某事。不要用被动结构。不要让无生命物体执行人类动作（如"投诉变成了解决方案"）。

4. **具体明确。** 不要用模糊断言（"原因是结构性的"）。说出具体的东西。不要用懒惰的极端词（"每个"、"总是"、"从不"）来糊弄。

5. **让读者身临其境。** 不要用旁观者视角。"你"胜过"人们"。具体胜过抽象。

6. **变化节奏。** 混合句子长度。两项胜过三项。段落结尾方式要多样化。不要用破折号。

7. **信任读者。** 直接陈述事实。省略软化、辩解和手把手引导。

8. **删除金句。** 读起来像"名言摘录"的句子，重写它。

## 快速检查清单

交付文章前，逐项检查：

- 有副词吗？全部删掉。
- 有被动语态吗？找到动作者，让他做主语。
- 无生命物体在做人类动词（如"决策浮现了"）？说出是谁做的。
- 句子以"什么/何时/哪里/为什么/如何"开头？重构它。
- 有"事情是这样的""关键在于"这类开场白？直接说重点。
- 有"不是X，而是Y"的对比结构？直接说Y。
- 连续三句长度相同？打破一句。
- 段落以短促金句结尾？换个方式。
- 有破折号吗？全部删掉。
- 模糊断言（"影响是深远的"）？说出具体影响是什么。
- 旁观者视角（"没有人设计过这个"）？让读者进入场景。
- 元衔接句（"本文剩余部分将解释……"）？删掉。让文章自己流动。

## 评分

每项1-10分：

| 维度 | 问题 |
|------|------|
| 直接性 | 是直接陈述，还是在做预告？ |
| 节奏感 | 是有变化，还是像节拍器一样机械？ |
| 信任度 | 是否尊重读者智商？ |
| 真实感 | 听起来像人写的吗？ |
| 密度 | 是否有能删掉的东西？ |

低于35/50：重写。
