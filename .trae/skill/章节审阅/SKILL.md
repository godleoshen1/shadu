# 章节审阅技能

## 功能说明
这个技能帮助你审阅写完的章节，检查是否符合规范，有无bug，逻辑是否合理，人物性格是否一致等。

## 何时使用
- 写完每章正文后，需要检查时
- 需要修改已写章节时
- 需要检查章节质量时

---

## 审阅清单（必须逐项检查）

### 1. 标题检查
- [ ] 标题有具体的意象、场景或情绪
- [ ] 不是四字成语式标题（风起云涌、龙争虎斗等）
- [ ] 不是空洞虚词堆砌
- [ ] 所有标题字数不同（与其他章节对比）
- [ ] 有画面感和节奏感

### 2. 字数检查
- [ ] 普通章：3000-4000字
- [ ] 高潮章：3500-4500字
- [ ] 全书核心章：4000-5000字
- [ ] 过渡章：2500-3000字
- [ ] 符合本章类型的字数要求

### 3. 开篇检查
- [ ] 开篇吸引人
- [ ] 不是平淡无奇的开头
- [ ] 快速进入主题
- [ ] 设定了本章基调

### 4. 核心事件检查
- [ ] 有明确的核心事件
- [ ] 核心事件推动剧情发展
- [ ] 不是流水账（起床→吃饭→打怪）
- [ ] 事件有意义，不空洞

### 5. 冲突检查
- [ ] 有关键冲突
- [ ] 冲突有张力，不简单
- [ ] 冲突解决合理
- [ ] 冲突推动剧情发展

### 6. 情绪检查
- [ ] 有明确的情绪基调
- [ ] 情绪有起伏（低→高→低或高→低→高）
- [ ] 情绪自然流畅，不刻意
- [ ] 情绪推动剧情发展

### 7. 节奏检查
- [ ] 不全程高潮（读者会累）
- [ ] 不全程平淡（读者会跑）
- [ ] 起承转合交替进行
- [ ] 节奏合适，考虑读者阅读体验

### 8. 结尾检查
- [ ] 结尾有钩子（悬念）
- [ ] 钩子让读者想看下一章
- [ ] 钩子合理，不刻意
- [ ] 与下一章衔接自然

### 9. 人物性格检查
- [ ] 人物性格一致，无OOC
- [ ] 人物行为符合其身份和背景
- [ ] 人物成长自然，有合理动机
- [ ] 对话符合人物性格

### 10. 世界观检查
- [ ] 严格遵循世界观设定
- [ ] 世界规则、地理与势力一致
- [ ] 力量体系按照设定的修仙境界划分
- [ ] 科技与修仙结合的设定合理

### 11. 逻辑检查
- [ ] 无逻辑漏洞
- [ ] 剧情发展合理
- [ ] 人物行为合理
- [ ] 前后一致，无矛盾

### 12. 内容质量检查
- [ ] 内容充实，不水文
- [ ] 不抄袭他人作品
- [ ] 逻辑清晰，不混乱
- [ ] 人物设定不崩坏
- [ ] 有创新，避免套路化

### 13. 对话检查
- [ ] 对话自然，不生硬
- [ ] 对话符合人物性格
- [ ] 对话有内容，推动剧情
- [ ] 避免无意义的闲聊

### 14. 描写检查
- [ ] 场景描写有画面感
- [ ] 人物描写生动
- [ ] 不过度描写，影响节奏
- [ ] 描写为剧情服务

### 15. 与细纲对照检查
- [ ] 符合细纲的核心事件
- [ ] 符合细纲的情绪基调
- [ ] 符合细纲的关键冲突
- [ ] 符合细纲的钩子（悬念）
- [ ] 没有偏离细纲的核心内容

## 审阅流程

### 第一步：通读章节
1. 先通读整章，了解整体内容
2. 记录第一印象
3. 注意明显的问题

### 第二步：逐项检查
1. 按照审阅清单逐项检查
2. 对每一项打勾或打叉
3. 记录发现的问题

### 第三步：详细分析问题
1. 对发现的问题进行详细分析
2. 确定问题的严重程度
3. 思考如何改进

### 第四步：提出修改建议
1. 针对每个问题提出具体的修改建议
2. 修改建议要具体、可操作
3. 提供修改示例（如果需要）

### 第五步：总结审阅结果
1. 总结主要问题
2. 给出整体评价
3. 提出下一步建议

## 问题分类

### 严重问题（必须修改）
- 逻辑漏洞
- 人物OOC
- 世界观设定矛盾
- 剧情严重偏离细纲

### 重要问题（建议修改）
- 节奏问题
- 情绪问题
- 对话问题
- 描写问题

### 轻微问题（可选修改）
- 个别错别字
- 语句不通顺
- 标点符号问题

## 注意事项
1. 审阅要客观，不要带有个人偏见
2. 问题要具体，不要太笼统
3. 修改建议要可操作，不要太模糊
4. 既要指出问题，也要肯定优点
5. 尊重作者的创作风格
6. 审阅后及时反馈给作者
7. 跟踪修改情况，确保问题解决
