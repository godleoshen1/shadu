# 小说架构创建技能

## 功能说明
这个技能帮助你系统地构建小说的完整架构，包括世界观、人物体系、修炼体系等。

## 何时使用
- 开始写小说前，需要搭建世界观框架
- 需要补充或修改世界观设定时
- 需要创建新的人物或势力设定时

## 如何使用

### 1. 创建小说总架构
**使用场景**：从零开始创建小说的整体架构

**工作流程**：
1. 先了解用户的小说类型、题材、字数要求
2. 创建作品定位（书名、类型、总字数等）
3. 设计核心冲突和主线大纲
4. 规划章节分布和节奏

**输出文件**：`01_世界观/00_小说总架构.md`

---

### 2. 创建世界规则书
**使用场景**：设定小说世界的基本规则

**工作流程**：
1. 设计修炼体系（境界划分、灵根类型等）
2. 设定资源体系（灵石、丹药、法宝等）
3. 规划功法、阵法、符箓等修炼要素
4. 加入主角专属的特色设定

**输出文件**：`01_世界观/01_世界规则书.md`

---

### 3. 创建地理与势力设定
**使用场景**：设定小说世界的地理环境和势力分布

**工作流程**：
1. 设计地理分区（大陆、区域、城市等）
2. 规划势力格局（宗门、组织、部落等）
3. 设定关键地点和秘境
4. 设计妖兽体系和经济资源体系

**输出文件**：`01_世界观/02_地理与势力.md`

---

### 4. 创建人物体系
**使用场景**：创建小说的所有人物设定

**工作流程**：
1. 设计主角（性格、背景、能力等）
2. 创建重要配角（女主、伙伴、导师等）
3. 设计反派体系（从底层到最终Boss）
4. 创建人物总表，方便管理和查阅

**输出目录**：
- `02_人物角色/01_主角/`
- `02_人物角色/02_重要配角/`
- `02_人物角色/03_反派/`
- `02_人物角色/04_其他角色/`
- `02_人物角色/00_人物总表.md`

---

## 写作规范要求

### 标题规范
- 标题必须有具体的意象、场景或情绪
- 禁止四字成语式标题（如风起云涌、龙争虎斗等）
- 禁止空洞虚词堆砌
- 所有标题字数必须不同

### 人物设定规范
每个人物设定必须包含：
- 基本信息（姓名、性别、年龄、身份、修为）
- 外貌特征
- 性格特点
- 能力与技能
- 重要经历
- 人际关系
- 目标与动机

### 架构一致性
确保所有设定前后一致，避免矛盾：
- 世界观与地理设定一致
- 人物性格与行为一致
- 修炼体系与能力设定一致

```
└── 03_剧情与进度/
    ├── 00_卷级细纲.md
    ├── 01_章节细纲.md
    └── 02_已写章节.md
```

## 注意事项
1. 先了解用户的具体需求，再开始创建
2. 保持设定的一致性和逻辑性
3. 遵循用户提供的写作规范
4. 定期检查和更新设定，避免矛盾
5. 确保文件命名规范，便于查找和管理
