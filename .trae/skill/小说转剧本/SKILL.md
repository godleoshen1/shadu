# 🎬 顶级AI漫剧分镜导演 · 专业技能包

## 技能概述

将任意长度的小说文本，转化为可直接投入AI视频生成工具（可灵/Runway/Pika/Luma/DALL-E/Midjourney/ComfyUI）的**工业级分镜剧本**。输出即复制、复制即使用。

---

## 一、核心能力体系

### 1.1 文本解构能力

| 分析维度 | 提取内容 | 视觉转化 |
|----------|----------|----------|
| 叙事功能 | 开场/冲突/高潮/转折/结局 | 确定镜头节奏快慢 |
| 情绪图谱 | 热血/虐心/悬疑/治愈/燃向 | 确定色彩与光影基调 |
| 高光标记 | 必须转化为视觉爆点的场景 | 分配特写/慢动作 |
| 角色动态 | 谁主导、谁弱势、谁在变化 | 确定角度仰俯关系 |
| 感官元素 | 声音/气味/温度/触感 | 转化为音效与画面质感 |

### 1.2 镜头语言能力

**景别策略（由远及近的叙事逻辑）**

| 景别 | 英文 | 画面范围 | 叙事功能 | 最佳时长 |
|------|------|----------|----------|----------|
| 极远景 | ELS | 人物极小，环境为主 | 建立世界观、孤独感 | 4-6秒 |
| 远景 | LS | 人物占画面1/4-1/3 | 展示环境与人物关系 | 3-5秒 |
| 全景 | FS | 人物全身 | 展示完整动作姿态 | 3-5秒 |
| 中景 | MS | 人物膝盖以上 | 日常对话、上半身动作 | 3-4秒 |
| 中近景 | MCU | 人物胸部以上 | 情绪表达、反应镜头 | 2-4秒 |
| 近景 | CU | 肩部以上/脸部 | 强化情绪、拉近距离 | 2-3秒 |
| 特写 | ECU | 局部（眼/手/嘴） | 极致冲击、强调细节 | 1.5-2.5秒 |
| 大特写 | XCU | 瞳孔/水滴/指尖 | 艺术感、情感爆破 | 1-2秒 |

```

### 4.2 处理流程

```
收到小说文本
│
▼
第1步：字数统计与节奏规划
├─ <500字 → 3-5个分镜
├─ 500-1500字 → 5-10个分镜
├─ 1500-5000字 → 10-20个分镜
└─ >5000字 → 分段处理，每段10-15个分镜
│
▼
第2步：文本解构
├─ 提取角色特征 → 建立角色设定卡
├─ 识别情绪曲线 → 规划镜头节奏
├─ 标记高光时刻 → 分配特写/慢动作
└─ 分析场景转换 → 设计转场方式
│
▼
第3步：分镜拆解
├─ 按叙事逻辑切分镜头
├─ 为每个镜头分配景别/角度/运动
├─ 设计光影与色彩方案
└─ 预估每个镜头时长
│
▼
第4步：提示词生成
├─ 使用标准公式组合提示词
├─ 确保角色特征一致性
├─ 加入动作与光影关键词
└─ 附上负面提示词
│
▼
第5步：输出剧本
├─ 按模板格式化输出
├─ 确保"复制即用"
└─ 附带使用说明
```

### 4.3 分镜数量决策树

```
开始
│
├─ 文本是高潮战斗场景？
│ └─ 是 → 增加30%分镜数量，缩短单镜时长
│
├─ 文本是情感细腻场景？
│ └─ 是 → 增加特写比例，延长单镜时长
│
├─ 文本是环境建立场景？
│ └─ 是 → 增加远景/全景比例
│
└─ 文本是对话场景？
└─ 是 → 中景+过肩+特写交替使用

公式：基础分镜数 = 字数 ÷ 100 × 1.2
最终分镜数 = 基础分镜数 × 场景系数（0.8~1.3）
```
